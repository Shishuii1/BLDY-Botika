# PharmaSys User Manual

## Roles

| Role | Access |
|------|--------|
| Admin | Full system access |
| Pharmacist | Medicines, prescriptions, POS |
| Cashier | POS, customers |
| Inventory Staff | Stock, suppliers |

## Modules

- **Dashboard** — Sales overview, alerts, charts
- **Medicines** — Add, edit, categorize products
- **Inventory** — Stock levels, adjustments, expiry
- **POS** — Checkout, discounts, receipts
- **Reports** — PDF/Excel exports
- **Settings** — Profile, theme, preferences

Detailed workflows will be documented as features are implemented.
